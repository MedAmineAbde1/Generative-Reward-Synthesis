        What Gets Migrated

        🔧

        SDF Version

        Updates the <sdf version> header from 1.6/1.7 to 1.11 and wraps bare <world> roots.

        🔌

        Plugin Migration

        Remaps 40+ classic lib*.so plugins to their gz-sim-*-system equivalents with correct name attributes.

        🌍

        World Plugins

        Auto-injects required Harmonic world plugins: Physics, UserCommands, SceneBroadcaster, Sensors, IMU, Contact, NavSat.

        📡

        Sensor Tags

        Converts ray → gpu_lidar, renames <ray> → <lidar>, removes sensor-level plugins, adds <topic> tags.

        ⚙️

        Physics Engine

        Migrates type="ode" → type="dart", removes ODE-specific solver/constraint tags with warnings.

        🎨

        Materials

        Converts Ogre <script> materials (Gazebo/Red, Blue, etc.) to <ambient>/<diffuse>/<specular> RGBA tags.

        🔗

        URIs & Paths

        Warns about model:// URIs requiring GZ_SIM_RESOURCE_PATH instead of GAZEBO_MODEL_PATH.

        🗑️

        Deprecated Tags

        Removes <sdf_version> in includes, invalid <static> on worlds, <allow_auto_disable>, and <ros> plugin sub-elements.
