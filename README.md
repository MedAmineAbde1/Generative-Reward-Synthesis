# Generative Reward Synthesis

**Bachelor Thesis Project**  
*Exploring generative methods for reward function synthesis in robotics*

---

## 📌 About the Project

This repository serves as the central hub for all my work related to my **Bachelor's thesis** on **Generative Reward Synthesis**. 

The goal of the project is to develop and experiment with generative approaches (likely using LLMs, diffusion models, or other generative techniques) to automatically synthesize reward functions for reinforcement learning agents in robotics.

## 🗂️ Repository Structure

- **`Code-Scraps/`** — Experimental code, notebooks, and miscellaneous scripts
- **`Gazebo_worlds/`** — Custom Gazebo simulation environments
- **`robot_model/`** — Robot URDF/SDF models and related configurations
- **`gz_migrator/`** — Tools for migrating between Gazebo versions
- **Various `.txt` files** — Notes on ROS2 architecture, topics, nodes, shortcuts, TODOs, etc.

## 🛠️ Technologies & Tools

- **ROS 2** (Robot Operating System)
- **Gazebo** (Simulation)
- **Python** & **Jupyter Notebooks**
- Reinforcement Learning
- Generative AI / LLMs (core focus of the thesis)
